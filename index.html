Biography

I am a PhD student in the School of Computing at the University of Utah in the robotics track of the Utah Robotics Center. I work in the Utah Learning Lab for Manipulation Autonomy (LL4MA) and am advised by Dr. Tucker Hermans. My research focuses on learning manipulation tasks from human demonstration with a targeted application to the problem of robust teleoperation over large latency.

I am looking for internship opportunities for Summer 2019 in robotics, machine learning, or software development. Please see my resume for my qualifications and take a peek at some of my coding projects for samples of my work.

Code Projects

If you are interested in any of my code, find bugs, or want additional features, please contact me. Not all of it is actively maintained, but if there is something you want to use, I am happy to pick up development on it again.

Robot Control in Gazebo

Robot control framework for simulations in Gazebo. It is designed to be extensible, allowing easy addition of new controllers, and it works in conjunction with my robot interface package to decouple the controllers from the particular robot you’re using.

Real-Time Robot Control

Real-time Orocos controllers for the KUKA LBR4+ robot. Includes a controller switching framework for safely swapping controllers at runtime, and a simulated FRI (Fast Research Interface) component that allows Orocos components to be tested Gazebo.

Robot Interface

Abstraction layer for controlling robots in a unified manner while accounting for robot platform-specific needs.

Teleoperation Mediated by Virtual Environment

A kinematic simulator with force rendering created with rViz to provide a training ground to learn robot policies from demonstration using a Phantom Omni haptic input device. The learned policy can then be executed on a real robot (or a simulated robot in Gazebo).

rViz Trajectory Visualization

Package for generating trajectory visualizations in rViz, including dynamic real-time visualizations showing a customizable trace of the robot’s end-effector.

Baxter Demonstration Recorder

A demonstration recorder for the Baxter robot utilizing the button and display interfaces to make it easier to record demonstrations when operating Baxter in gravity compensation mode.

Data Logging

Two data logging frameworks for recording robot and sensor data. One uses HDF5 files with an h5py interface, and the other post-processes rosbag data into Pandas dataframes.

Baxter Multimaster

A fork of the multimaster package which I augmented to allow teleoperation of a slave Baxter robot by moving the limbs of master Baxter robot in gravity compensation mode.
